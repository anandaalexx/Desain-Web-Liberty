# Liberty Fishing Pond Website

Website ini dikembangkan sebagai media informasi dan promosi untuk Liberty Fishing Pond yang berlokasi di KM 29 Samboja.

## Tujuan Pengembangan

Website ini bertujuan untuk membantu pengunjung mendapatkan informasi mengenai tempat pemancingan Liberty serta mempermudah proses reservasi. Selain itu, sistem ini juga dirancang untuk mendukung pendaftaran kegiatan atau lomba yang diadakan oleh pengelola.

## Cara Menjalankan Proyek

Untuk menjalankan proyek secara lokal, gunakan perintah berikut:

```bash
npm start
```
